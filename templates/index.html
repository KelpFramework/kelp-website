What is Kelp?
Kelp is a large brown algae seaweed that is found in the ocean.
Just kidding!

Kelp is a framework built for version-independent spigot plugin development. It is made to increase readability to a maximum, while reducing maintenance costs to a minimum. This enables you to make your plugins compatible with all spigot versions from 1.8 to 1.16 with only one single codebase.

Features
view_sidebar Sidebar
Stop using nasty workarounds to display & update your sidebars. Use a simple and component-based approach that works just like other familiar frameworks such as React or Flutter. Updates can now be done with ease as you can distinguish between stateful and stateless components. Animated titles are supported out of the box with a bunch of animation algorithms ready to be used.
See on GitHub
widgets Inventories
Create stunning inventories with animations and widget-based design. Instead of just putting items into an inventory, you put in widgets handling calculations that are needed to create a pagination or toggleable buttons/... You can create your own widgets as well!
Not on GitHub yet
event_note Events
Listen to new custom events that are called when a player changes their language for example. Never forget to register listeners again and create dynamic listeners on server runtime for a more functional approach.
See on GitHub
celebration Particle Engine
Create stunning particle effects based on mathematical templates for particle calculation and create your own particle algorithms with ease. All particles you send are version-independent of course!
Not on GitHub yet
schedule Schedulers
A complete redesign of the Bukkit scheduler library simplifying asynchronous operations and especially synchronization with the server's main thread to handle non-thread-safe operations easier.
